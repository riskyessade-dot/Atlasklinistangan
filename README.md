# Atlasklinistangan

This repository contains a small Vite + React app: Atlas Klinis Tangan — an interactive reference and quiz for hand conditions.

How to run locally

1. Install dependencies

   npm install

2. Run development server

   npm run dev

3. Build for production

   npm run build

Deploy to Vercel

1. Push this repository to GitHub (already done).
2. Go to https://vercel.com/new and import the repository `riskyessade-dot/Atlasklinistangan`.
3. Vercel should detect a Vite React app automatically. If prompted, use:
   - Build command: `npm run build`
   - Output directory: `dist`

After deploy, Vercel will provide a live URL for your app.

If you want, I can also add a GitHub Actions workflow or a Netlify config, or tweak the UI further.
